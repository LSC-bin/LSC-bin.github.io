# AI ClassBoard

실시간 학습 플랫폼

## 📁 프로젝트 구조

```
├── src/                    # 새 React 앱 소스 코드
├── prototype/              # 기존 프로토타입 파일들
├── docs/                   # 문서
├── config/                 # 설정 파일들
└── e2e/                    # E2E 테스트
```

## 🚀 시작하기

### 기존 프로토타입 사용
```bash
# prototype/index.html을 브라우저에서 열기
# 또는 정적 서버 실행
python -m http.server 8000
```

### 새 React 앱 사용
```bash
npm install
npm run dev
```
